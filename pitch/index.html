SYSTEM ARCHITECTURE
LIVE
🏗️ ARCHITECTURE OVERVIEW
THREE PIPELINES. ONE LIVE ROOM.
Pipeline 01 — State
📊
REST + WebSocket Event Bus
Frontend → GET /api/sessions
Firestore → Snapshot Listener
Gateway → Push via ws://

Every agent action instantly syncs to the UI. No polling. No lag.
Pipeline 02 — Voice
🎙️
Bidirectional Audio Stream
Chairman Mic → LiveKit
ElevenLabs STT → Transcript
Z.AI GLM → Agent Response
ElevenLabs TTS → Audio Out
Pipeline 03 — Agent Isolation
🔒
Isolated Memory + Shared Board
Each agent has private memory. They share only the Crisis Board. No agent can read another's internal reasoning — just like real advisors in separate rooms.
